energy_data_visualization.global_var.path_folders module¶

Definition of the folders to read and save the data and the folders where the plots should be saved. Although a default configuration is proposed, the user can decide her own folders by creating a module pers_var.py in this folder.

energy_data_visualization.global_var.pers_var module¶

Non-synchronized file to set the folders where the raw data are saved, the transformed data should be saved and the plots and the outputs should be saved.
